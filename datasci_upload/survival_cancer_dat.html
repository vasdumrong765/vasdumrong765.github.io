Brief Concepts of Survival Analysis in Clinical Oncology

Time-to-event analysis or survival analysis aims to estimate the expected duration of time until one or more events occur. This is to be contrasted with logistic regression which aims to answer the question whether the event will occur.

In the context of clinical oncology, time-to-event may be any of the following events:

  • time to death
  • time to response to treatment
  • time to relapse/reoccurrence (i.e. remission duration)

However, an individual may leave the study prior to study completion. Therefore, we do not know the answer whether that an event occurs for that patient given the study duration. This type of data is referred to as right-censored. Some examples of right-censored scenarios include:

  • Fail to follow up
  • Withdrawn from study
  • Disease progression (opposite effects)
  • Event of interest did not occur

Thus, in survival analysis, we do not need all individuals to be present until the completion of the study. Rather, censored observations are used in the dataset up to the time they are censored.

Theory

Here, we define survival time \(T\) (or failure time) as a continuous non-negative random variable, representing a time duration from one point to an event of interest.

  1. Survival function \(S(t)\)

Suppose \(F(t)\) is a cumulative distribution function for the probability of observing \(T\) less than or equal to a specified time \(t\). \[F(t) = P(T \leq t) = \int_{0}^{t}{f(s)\cdot ds}\] Then, the survival function is defined as the probability of observing \(T \geq t\) which is also the complement of \(F(t)\) with the initial condition \(S(t=0)=1\) and at \(S(t \rightarrow \infty) = 0\) \[S(t)=P(T \geq t) = 1 - F(t)\]

  1. \(S(t)\) is characterized by the hazard rate function \(h(t)\)

\(h(t)\) is an instantaneous failure rate (other names: intensity function, force of mortality). In fact, one may say that the shape of \(S(t)\) depends on \(h(t)\). Although understanding \(h(t)\) is quite intuitive, the derived relationship between \(h(t)\) and \(S(t)\) can be complicated and will be skipped in this notebook. \[\lambda(t) = h(t) = \frac{f(t)}{S(t)} = -\frac{d}{dt}\ln(S(t))\] Note that the common definition of \(h(t)\) is usually written in the limit form. Using Bayes’ rule and the differential relationship between PDF (\(f(t)\)) and CDF (\(F(t)\)), you can derive the equation above derivation here. \[ \lambda(t) = \lim_{dt \rightarrow \infty} \frac{P(t \leq T <t+dt|T \geq t)}{dt}\] 3. Cumulative hazard function \(H(t)\) can be derived from integration over \(dt\) \[ H(t) = \int_{0}^{t} h(u) \cdot du = -\ln(S(t))\] Thus, we can write the survival function in terms of cumulative hazard function as: \[ S(t) = e^{-H(t)}\] 4. Mean survival is the expected value of survival time

\[\mu = \bar{T} = E(T) = \int_{0}^{\infty}tf(t)\cdot dt = \int_{0}^{\infty}S(t)\cdot dt\] Overall, think of \(S(t)\) as the not-failing probability and \(h(t) \geq 0\) as a conditional failing rate with no upper bound.

1. Non-parametric method: Kaplan-Meier

The function that is used to calculate the Kaplan-Meier estimate is survfit(formula, data). The probability of survival at time \(t\) given by the KM estimator is: \[ \widehat{S}_{KM}(t)=\prod_{j:\tau _j<t}\left ( 1-\frac{d_j}{r_j} \right ) \] * \(\tau _j\) is the \(j^{th}\) distinct death time
* \(d_j\) is the number of death at time \(\tau _j\)
* \(r_j\) is the number of patients who were at risk at time \(\tau _j\). Note this also includes those who died at \(\tau _j\).

Then, we can use the log-rank test the null hypothesis that all survival curves are the same. It is based on the Chi-square statistics.

2. Parametric model usually refers to as accelerated failure time (AFT) model

The use of a parametric method is convenient when we are interesting in performing a regression so that we can model the effect of covariates that may affect the inferences of the outcome variable.

Exponential hazard function

In parametric method, we model the outcome variable based on a statistical distribution. The simplest case here is the exponential distribution, where the hazard function is constant (i.e. constant event rate). Although it is generally too simplistic for human lifetimes. \[ h(t) = \lambda \] \[ S(t) = e^{-\lambda t} \] \[ f(t) = h(t)\cdot S(t)=\lambda e^{-\lambda t} \] \[ E(T) = \int_{0}^{\infty}S(t) \cdot dt = \frac{1}{\lambda}\] Now, suppose that we have a dataset of survival \(T\) with censorship, assumed to follow a distribution of unknown parameter \(\lambda\). One way to estimate such parameter is to used the maximum likelihood estimator which can handle censored data well with an indicator variable (\(\delta=(0,1)\)). Below is a likelihood function: \[ L(\lambda|T_1, T_2, ...,T_n) = \prod_{i=1}^{n}h(t_i, \lambda)^\delta_{i}\cdot S(t_i,\lambda)^{1-\delta} \] We can obtain a point estimate of \(\lambda\), briefly, by maximizing the likelihood function. But this is difficult while log-transformation \(l\) makes it simpler. Calculus tells us that we can solve for \(\lambda\) that maximizes \(L\) by setting its first derivative equal to 0 (scoring function). On the other hand, the second derivative of this function yields an information function \(l = -I(\lambda)\). From statistical theory, variance of \(\hat\lambda\) is approximately equal the inverse of information. This can then be used to carry out hypothesis test and CI. \[\hat{Var}(\hat\lambda) \approx I^{-1}(\hat\lambda)\]

# fit exponential model
s2_exp = survreg(Surv(time_d, status) ~ treatment + sex + age + race, data=data_df, dist = "exponential")

Weibull distribution

More realistic distributions that can be used in the same manner with survreg are (“weibull”, “exponential”, “gaussian”, “logistic”,“lognormal”, or “loglogistic”). For Weinbull distribution, the survival function is characterized by: \[ h(t) = \alpha \lambda (\lambda t)^{\alpha -1} = \alpha \lambda^{\alpha} t ^{\alpha - 1} \] \[ S(t) = e^{-\alpha(\lambda t )} \]

4. Semi-parametric Cox model

In Cox proportional hazard model, the hazard function is modeled in 2 parts:

  • the baseline hazard function \(h_0(t)\)
  • the time-independent parametric components assuming multiplicative effects of covariates. Mathematically, this is convenient because coefficients are linearly model in the logarithmic scale, allowing hazard ratio computation.

Cox model does not assume the shape of the baseline \(h_0(t)\). Therefore, it is not a full parametric model. The parametric part comes from regression-like behavior of the coefficients of the covariates.

\[ h(t|X=x)=h_0(t)e^{x^{T}\beta} \] \[ H(t|x) = e^{x^{T}\beta} \int_{0}^{t}h_0(s) \cdot ds = e^{x^{T}\beta} \cdot H_0(t) \] \[ S(t|x) = e^{x^{T}\beta} \cdot S_0(t) \] The hazard ratio is an estimate of the ratio of the hazard rate in the treated versus the control group (or a reference group). A HR < 1 indicates reduced hazard of death whereas a HR > 1 indicates an increased hazard of death. It is not a probabiliy, it’s a ratio between rates.
